# NOMADS Coffee Shop Landing Page

NOMADS is a modern, cinematic coffee shop landing page concept for Hinigaran, Negros Occidental.  
It features a warm editorial aesthetic, animated hero sections, responsive navigation, and polished section transitions.

## Repo Description (for GitHub)

Use this short description on your GitHub repository:

`Cinematic, responsive coffee shop landing page built with HTML + Tailwind CDN for NOMADS (Hinigaran, Negros Occidental).`

## Features

- Responsive single-page layout
- Animated hero text and section reveal effects
- Mobile-friendly navigation menu
- Menu, vibe/gallery, and location sections
- Custom color palette and typography styling

## Run Locally

Because this is a static site, you can open `index.html` directly in a browser.

If you prefer a local server:

```powershell
cd c:\laragon\www\nomads
php -S localhost:8000
```

Then open `http://localhost:8000`.

## Customization

- Main layout/content: `index.html`
- Images and branding assets: `assets/img/`
- Colors, animation, and typography are configured inline in the Tailwind config/script inside `index.html`
